# ת״ת אוהל תורה - מערכת ניהול קמפיין גיוס

מערכת לניהול קמפיין גיוס כספים, כולל סנכרון עם נדרים פלוס ו-Google Sheets.

## פריסה ל-Netlify

1. חיבור ה-repo ל-Netlify – Build command ריק, Publish directory: `.`
2. הגדרת משתני סביבה (אם משתמשים ב-Nedarim proxy): `NEDARIM_API_PASSWORD`, `NEDARIM_MOSAD_ID`

## Google Apps Script

העתק את `SheetsSync.gs` או `PASTE_THIS_IN_APPS_SCRIPT.gs` ל-Google Apps Script ופרס את ה-Web App.
